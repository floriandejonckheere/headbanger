# Headbanger UI

Headbanger is a music recommendation and discovery platform.

## Installation

Install NPM packages:

```
yarn install
```

## Development

Run the Vue development server:

```
yarn serve
```

## Production

Build the project:

```
yarn build
```

## Testing

Run the test suite:

```
yarn test:unit
```

### Lints and fixes files
```
yarn lint
```
